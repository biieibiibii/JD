  不再维护，仅当作学习记录知识点
## 知识点
### 滑块验证码自动验证
  **破解思路：**
  
  1.使用**鼠标轨迹记录方法**对滑块验证界面进行手动记录数据。

  2.使用浏览器自动化库提前获取对应的页面**url，小滑块，验证码背景图**的数据。

  3.使用图像识别技术判断需要的滑动距离。
  
  4.选取合适的记录数据进行模拟手动拖动以完成验证码。

  只要手动记录够精准，滑块验证码的破解率可达到100%。
### 图像识别的初次使用
  图像识别主要用于验证码的距离判定和通知模块的人物头像识别。
### 多线程的初次使用
  多线程主要用于分离京东的抓取数据和通知系统，由通知系统可随时影响数据抓取系统的启停。
### 浏览器自动化
  主要点在于正确的获取cookies，以便于解决重复登录问题。
### 网络api数据抓取
  先从浏览器网络监控的数据中获取相应的数据url，根据api监控方法去解决**数据获取，登录验证，验证码页面判断等决策**
### 通知模块(微信通知)
  采用鼠标模拟操作，对发过来的微信消息转化为对应的程序步骤，无需注册云服务器就可以实现远程控制。

  当然，这样识别消息必然会**导致cpu使用功率与获取消息时间间隔成反比**。
